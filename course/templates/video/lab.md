# Video N Mini-Lab: Title

## Setup

List local files and tools. Do not require a network connection unless the lesson explicitly needs one.

## Command

Give the exact command from the repository root.

## Prediction

Ask the learner to write down an expected result before running the command.

## Steps

Give numbered, independently checkable actions.

## Expected Output

Show the exact stable output, separating it from any environment-dependent output.

## Explanation

Connect each important output line to the learning objective.

## Extension

Ask for one small change and one sentence explaining the result.
